Axioms of Real Analysis

  • $\mathbb{R}$ is an ordered field
  • $\mathbb{R}$ contains $\mathbb{Q}$ as a subfield
  • Axiom of Completeness: Every nonempty set of real numbers that is bounded above has a least upper bound.

▷

Nested Interval Property

For each $n \in \mathbb{N}$, assume we are given a closed interval $$I_n = [a_n, b_n] = \{x \in \mathbb{R}: a_n \leq x \leq b_n \}$$

Assume also, that each $I_n$ contains $I_{n+1}$. Then the resulting sequence of closed intervals $$I_1 \supseteq I_2 \supseteq I_3 \supseteq ...$$ has a non-empty intersection, i.e. $\cap_{n=1}^\infty I_n \neq \phi$

Archimedean Property

  • Given any number $x \in \mathbb{R}$, there exists an $n \in \mathbb{N}$ satisfying $n > x$
  • Given any real number $y > 0$, there exists an $n \in \mathbb{N}$ satisfying $\frac{1}{n} < y$

Density of $\mathbb{Q}$ in $\mathbb{R}$

For every two real numbers $a$ and $b$ with $a < b$, there exists a rational number r satisfying $a < r < b$

Size of $\mathbb{Q}$ and $\mathbb{R}$

  • The set $\mathbb{Q}$ is countable
  • The set $\mathbb{R}$ is uncountable

Cantor's Theorem

Given any set $A$, there does not exist a function $f: A \to P(A)$ that is onto.

▷

Dedekind's Construction of $\mathbb{R}$ from $\mathbb{Q}$

A subset $A$ of the rational numbers is called a cut if it posseses the following three properties:

  • $A \neq \phi$ and $A \neq \mathbb{Q}$
  • If $r \in A$, then $A$ also contains every rational $q < r$
  • $A$ does not have a maximum, i.e., if $r \in A$, then there exists $s \in A$ with $r < s$

The real numbers $\mathbb{R}$ are then defined to be the set of all cuts in $\mathbb{Q}$.

▷

# Useful Results

If $A \subseteq B$ and $B$ is countable, then $A$ is either countable, finite or empty

  • If $A_1, A_2, ..., A_m$ are each countable sets, then the finite union $A_1 \cup A_2 \cup ... \cup A_m$ is countable.
  • If $A_n$ is a countable set for each $n \in \mathbb{N}$, then $\cup_{n=1}^\infty A_n$ is countable.

# Definitions

Function

Given two sets $A$ and $B$, a function from $A$ to $B$ is a rule or mapping that takes each element $x \in A$ and associates it with a single element of $B$. In this case, we write it as $f: A \to B$.

A function $f: A \to B$ is a one-one function if $a_1 \neq a_2$ in $A$ implies that $f(a_1) \neq f(a_2)$ in $B$.

A function $f: A \to B$ is onto if given any $b \in B$, it is possible to find an element in $a \in A$ for which $f(a) = b$

▷

Dirichlet Function

$$g(x) = \begin{cases} 1 & \text{ if } x \in \mathbb{Q}\\ 0 & \text{ if } x \notin \mathbb{Q} \end{cases}$$

Triangle Inequality

The absolute value function satisfies $|a+b|\leq |a| + |b|$ for all choices of $a$ and $b$.

This inequality is often used as: $$ \begin{aligned} |a-b| & = |(a-c)+(c-b)| \\ & \leq |a-c| + |c-b| \end{aligned} $$

Least Upper Bound

A real number $s$ is the least upper bound for a set $A \subseteq \mathbb{R}$ if it meets the following two criteria:

  • $s$ is an upper bound for $A$
  • if $b$ is any upper bound for $A$, then $s \le b$

OR

A real number $s$ is the least upper bound for a set $A \subseteq \mathbb{R}$:

  • if $s$ is an upper bound for $A$
  • iff for every choice of $\epsilon > 0$, there exists an element $a\in A$ satisfying $s-\epsilon < a$

Two sets have the same cardinality if there exists $f: A \to B$ that is one-one and onto. In this case, we write $A ~ B$

A set $A$ is countable if $\mathbb{N} ~ A$. An infinite set that is not countable is called uncountable
